# Hotel-System
Win32控制台应用程序-简单的酒店预约系统

完成语言：C++

需要软件：Microsoft Visual Studio

主要功能：
酒店信息查询；酒店预约；解除预约等

运行方式：
下载后直接双击HotelSystem.sln即可运行
